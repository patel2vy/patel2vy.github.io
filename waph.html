Lab-1 Overview

In Lab 1, we will explore the HTTP protocol and web application programming in two parts. Part I focuses on learning how to use Wireshark for HTTP traffic analysis and how to interface with HTTP servers using telnet. The goal is to comprehend HTTP components, methods, and replies. Part II focuses on web application development, with an introduction to the Common Gateway Interface (CGI) and the implementation of a small web application in C that handles user input. We will also look into web application development using PHP, focusing on the differences between HTTP GET and POST requests, as well as the security implications. This extensive lab attempts to provide us a practical grasp of HTTP, CGI, and fundamental web application development.

Lab-2 Overview

In Lab 2, we must accomplish two assignments related to front-end web development. The first task will be to grasp the fundamentals of HTML and forms. Following that, we will look at JavaScript and the different techniques for incorporating it in an HTML file. Task 2 will cover AJAX, CSS, jQuery, and the integration of web APIs utilizing public APIs, giving you hands-on experience with front-end web development principles.

Hackathon-1 Overview

TThis hackathon aims to raise awareness about cross-site scripting (XSS) threats. Participants learn about code vulnerabilities, follow OWASP rules, and use safe coding methods to guard against XSS assaults. The laboratory consists of two assignments. Task 1 involves attacking the URL http://waph-hackathon.eastus.cloudapp.azure.com/xss/, which has six degrees of susceptibility to XSS assaults. Task 2 entails preventing XSS attacks by adopting secure coding techniques such as input validation and output sterilization.
